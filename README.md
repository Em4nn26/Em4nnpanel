## ❗ Troubleshooting

**Permission denied**

```bash
chmod +x em4nn26.sh
```

**File not found**

```bash
sudo apt install dos2unix
dos2unix em4nn26.SH
chmod +x em4nn26.SH
./em4nn26.SH
```

**Run manually**

```bash
bash em4nn26.sh
```
